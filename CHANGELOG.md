# Changelog

All notable public changes to `dp-release-card` will be documented here.

This project uses a small pre-1.0 version line. JSON artifact fields may change
before 1.0, but changes should be explicit and tested.

## 0.1.0 - 2026-06-17

- Added strict CSV histogram release flow.
- Added finite-precision `discrete_laplace` mechanism.
- Added HMAC-SHA256 signed receipts.
- Added receipt verification and optional release digest verification.
- Added Markdown release cards.
- Added strict input, JSON, schema, and output-path validation.
- Added CI on Python 3.10, 3.11, and 3.12.
- Added public artifact schemas and committed example outputs.
- Added packaging checks, issue templates, PR template, release checklist, and roadmap.
